BET33 PNG Logo Patch
====================
이 패키지는 로고를 PNG 전용으로 교체하고, 크기를 56px로 확대합니다.

적용 방법
1) 여러분의 프로젝트 루트에 있는 파일을 이 ZIP의 파일로 덮어쓰기 하세요.
   - index.html
   - assets/logo.png
2) 브라우저에서 강력 새로고침(Ctrl/Cmd+Shift+R) 후 확인합니다.
3) 더 크게 원하시면 index.html 내 CSS에서 height 값을 84px 등으로 변경하세요.

주의
- 경로와 대소문자를 정확히 유지하세요: assets/logo.png
- 기존 SVG 참조가 있다면 모두 PNG 경로로 통일했습니다.
